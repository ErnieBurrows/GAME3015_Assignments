Detailed Description

First-person camera implementation for 3D scene viewing.

Manages camera space orientation and perspective projection parameters. Automatically handles view matrix updates when camera parameters change.

◆ Camera()

Camera::Camera ( )

Constructs a camera with default perspective parameters.

Initializes camera with a 45-degree FOV, 1:1 aspect ratio, and 1-1000 view volume.
