\doxysubsection{Detailed Description}
Direct3D 12 application base class. 

Provides core functionality for Direct3D 12 applications including\+:
\begin{DoxyItemize}
\item Window management
\item Direct3D device and swap chain initialization
\item \doxylink{struct_command}{Command} queue/allocator management
\item Resource descriptor heaps
\item Main application loop 
\end{DoxyItemize}
